# 🏦 Bank Management System

A console-based Bank Management System developed in C that allows users to manage bank accounts efficiently using file handling. The system provides essential banking operations such as account creation, deposits, withdrawals, balance inquiry, and transaction history.

## 📌 Features

- Create a new bank account
- Deposit money
- Withdraw money
- Check account details
- View account balance
- File-based data storage
- Transaction logging
- Export account data

## 🛠 Technologies Used

- C Programming
- File Handling
- Structures
- Functions
- VS Code
- Git & GitHub

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/anchalicapatil2007-cell/Bank-Management-System.git
```

2. Compile

```
gcc banking1.c banking2.c -o bank
```

3. Run

```
./bank
```
